# Audio Files - Vương Lâm Portfolio

Thêm các file âm thanh vào các thư mục tương ứng:

## Ambient (Nhạc nền môi trường)
- `ambient/night-wind.mp3` - Tiếng gió đêm, tiếng côn trùng... (loop)

## Music (Nhạc nền chính)
- `music/main-theme.mp3` - Nhạc nền Tiên Hiệp, epic orchestral (loop)

## SFX (Hiệu ứng âm thanh)
- `sfx/jump.mp3` - Tiếng nhảy (whoosh ngắn, ~0.3s)
- `sfx/land.mp3` - Tiếng đáp đất (thump nhẹ, ~0.2s)
- `sfx/sword-whoosh.mp3` - Tiếng kiếm rít gió (metallic whoosh, ~0.5s)
- `sfx/phoenix-cry.mp3` - Tiếng phượng hoàng kêu (bird cry + fire, ~1s)
- `sfx/ui-click.mp3` - Tiếng click UI (soft click, ~0.1s)
- `sfx/unlock.mp3` - Tiếng mở khóa (magical chime, ~0.5s)
